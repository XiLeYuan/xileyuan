# 结婚吧 - 颜色系统设计建议

## 主题色（已确认）
- **温暖珊瑚红** `#FF6B6B` - 传达温暖、热情、活力、爱情

---

## 一、认证图标颜色系统

### 1. 普通用户认证
**推荐颜色：信任蓝**
- **主色**：`#4A90E2` (信任蓝) - 传达可信、专业、真实
- **浅色**：`#6BA3E8` (用于浅色背景)
- **深色**：`#357ABD` (用于深色背景)

**设计理由**：
- 蓝色在用户认知中代表信任、可靠
- 与珊瑚红形成冷暖对比，视觉层次清晰
- 适合普通用户认证，传达"真实可信"的感知

### 2. 红娘认证
**推荐颜色：专业深蓝**
- **主色**：`#2C5F8D` (专业深蓝) - 传达专业、权威、资深
- **浅色**：`#4A7BA8` (用于浅色背景)
- **深色**：`#1E4A6B` (用于深色背景)

**设计理由**：
- 深蓝色比普通认证更深，体现红娘的专业性和权威性
- 与普通认证形成层次区分
- 传达"专业服务"的感知

**替代方案（可选）：**
- **专业绿** `#4ECDC4` - 如果希望更温和，可用现有的薄荷绿
- **深绿** `#2E9E96` - 更深层次的绿色，保持专业感

---

## 二、VIP图标颜色系统

### 推荐颜色：尊贵金
- **主色**：`#FFB400` (温暖金) - 传达尊贵、价值、特权
- **浅色**：`#FFC933` (用于浅色背景)
- **深色**：`#E6A200` (用于深色背景)
- **渐变**：`#FFD700` → `#FFA500` (金色渐变，用于卡片背景)

**设计理由**：
- 金色在用户认知中代表尊贵、VIP、特权
- 与珊瑚红形成暖色调搭配，和谐统一
- 视觉冲击力强，易于识别

**当前项目已有金色系，建议统一使用：**
- `#FFD700` (标准金)
- `#FFA500` (深金)
- `#FFB400` (温暖金)

## 三、标签颜色系统

### 1. 精选/推荐标签
- **背景色**：`#FF6B6B` (温暖珊瑚红) - 与主题色一致
- **文字色**：`#FFFFFF` (白色)

### 2. 普通信息标签
- **背景色**：`#F0F0F0` (浅灰) - 中性、不抢眼
- **文字色**：`#666666` (中灰)

### 3. 特色标签（如：高颜值、高学历等）
- **背景色**：`#FFF4E6` (浅橙) - 温暖、突出
- **文字色**：`#FF8C42` (橙红)

### 4. 状态标签
- **在线**：`#4ECDC4` (薄荷绿) - 清新、活跃
- **离线**：`#CCCCCC` (浅灰) - 中性
- **新用户**：`#FFD93D` (温暖黄) - 醒目、新鲜

### 5. 分类标签
- **学历**：`#E3F2FD` (浅蓝) + `#1976D2` (深蓝文字)
- **收入**：`#F3E5F5` (浅紫) + `#7B1FA2` (深紫文字)
- **地区**：`#E8F5E9` (浅绿) + `#388E3C` (深绿文字)

## 七、使用建议

### 视觉层次
1. **第一层（最重要）**：温暖珊瑚红 - 主要操作、精选内容
2. **第二层（重要）**：信任蓝/专业深蓝 - 认证标识
3. **第三层（强调）**：尊贵金 - VIP特权
4. **第四层（辅助）**：薄荷绿、温暖黄 - 状态、标签
5. **第五层（中性）**：灰色系 - 次要信息、禁用状态

### 搭配原则
- **暖色搭配**：珊瑚红 + 金色 + 橙色（和谐统一）
- **冷暖对比**：珊瑚红 + 蓝色（层次清晰）
- **互补色**：珊瑚红 + 薄荷绿（活力清新）

### 可访问性
- 确保文字与背景对比度 ≥ 4.5:1（WCAG AA标准）
- 重要信息使用高对比度颜色
- 考虑色盲用户，使用形状+颜色双重标识
